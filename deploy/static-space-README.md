# 🏍️ Bikefinder RAG — project showcase

Agentic RAG over **32,395 motorcycles (1894–2024)** and **107,952
embedded owner comments** (bikez.com), enriched with French launch
prices (motoplanete), NHTSA safety recalls and an indicative used-price
cote — no LangChain, raw tool-use loop, PostgreSQL + pgvector + BGE-M3.

This static Space hosts the project's proof documents, layered
evaluation results and a real captured session of the agent. Source and
run-it-yourself instructions:
[github.com/maax6/bikefinder-rag](https://github.com/maax6/bikefinder-rag).
