Captured 15 July 2026 from the local demo (Ollama, mistral-small3.2) — answers unedited except the last list, shortened for space. French questions retrieve English forum data: the embeddings are multilingual by design. Some category labels come noisy from the source (the trail bikes in the last answer are bikez.com's own « Super motard » labels); the agent is instructed to flag intruders, and a curated French category layer now cross-checks them.

Run it yourself

The live demo runs locally (visitors bring an Anthropic or OpenRouter key, or a local Ollama model — no key at all). Everything ships in the repo, datasets included:

git clone https://github.com/maax6/bikefinder-rag — then follow the README (one docker run for Postgres, one loader script, one command to chat).
